CareerLens — Full Plan
======================

WHAT IT IS:
  AI career research buddy with a Picasso-style canvas chat interface.
  You tell it your dream role, it has a conversation with you, takes your resume,
  then does deep research across the web and builds a Pinterest-style vision board.

TECH STACK:
  Frontend:  Next.js 16 + React 19 + Tailwind (Vercel)
  Backend:   Python FastAPI (Railway/Fly.io)
  AI:        OpenAI GPT-4o
  Search:    Serper.dev (Google search)
  Transcripts: youtube-transcript-api (free, no key)
  Database:  Supabase (later — for persistence, auth, board memory)

API KEYS NEEDED (none set up yet):
  [ ] OpenAI API Key — chat, resume parsing, synthesis ($5 minimum to start)
  [ ] Serper.dev API Key — web search (free tier: 2,500 searches/mo, no card needed)
  [ ] YouTube Data API Key — NOT NEEDED (using free transcript library instead)
